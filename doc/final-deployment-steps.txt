**Register webhook with Stripe**

For development the Stripe CLI (https://stripe.com/docs/stripe-cli) is used to relay webhook calls to the localhost Django server.  For production the webhook endpoint needs to be set up on Stripe dashboard

Add an endpoint to Stripe (URL of Heroku app)/checkout/wh/

Copy Stripe public key, secret key and webhook signing secret/signing key to Heroku Config Vars matching the os.environ variables expected by settings.py

**Set up email sending using gmail**

Log into your google account

Create an app password (under 2-step verification).

In the Heroku Config Vars, set EMAIL_HOST_USER to your gmail address and EMAIL_HOST_PASS to the app password generated by google.
